# bulb_example
install qt5 libs for this project for this run commands 
```
sudo apt -y install qml-module-qtquick-controls
sudo apt -y install qml-module-qtquick

```
 
# How to build


```
mkdir build
cd build 
cmake ..
```

# To run
in parent directory 
```
roscore
in second terminal
cd devel/lib/bulb
./bulb
```
